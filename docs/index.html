Breadcrumbs

Breadcrumbs can be found on each page except the homepage. Since it is used for navigation we use the <nav> element.

home > oplossingen 
<nav class="breadcrumb" role="navigation">
    <div class="wrapper">
      <span class="item first"><a href="/">home</a></span>
      <span class="separator">></span>
      <span class="item"><a href="/oplossingen">oplossingen</a></span>
    </div>
  </nav>

Buttons

Button styles can be applied to any element. Typically you'll want to use either a <button> or an <a> element:

Don't click 
<button class="button">Click</button>
<a class="button" href="http://www.goalgorilla.com">Don't click</a>
<span class="readmore-arrow"></span>

Node compact

Compact nodes are used to display on overview of nodes, like news and services.

Service title

Bekijk deze oplossing 
<div class="view-solutions">
  <div class="views-row">
    <a href="#" class="node-compact">
      <header>
        <h2> Service title </h2>
      </header>
      <article>
        <span class="node-button-area"><span class="readmore-arrow"></span> <span class="button"> Bekijk deze oplossing </span></span>
      </article>
    </a>
  </div>
</div>
